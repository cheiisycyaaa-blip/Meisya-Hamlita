Judul : 
Web Monitoring Jaringan Laboratorium Teknik Komputer.
Deskripsi Singkat Website :
Website Monitoring Jaringan Laboratorium Teknik Komputer merupakan website sederhana yang digunakan untuk mencatat dan memonitor kondisi jaringan komputer di laboratorium. Website ini memungkinkan mahasiswa atau teknisi laboratorium untuk melaporkan gangguan jaringan seperti koneksi lambat, tidak terhubung ke internet, atau perangkat jaringan bermasalah. Website dibangun menggunakan HTML sebagai struktur halaman, CSS untuk tampilan, JavaScript untuk validasi input, serta PHP untuk memproses dan menampilkan data laporan jaringan.
Daftar Fitur Utama  :
Form laporan gangguan jaringan
Input lokasi lab dan jenis gangguan
Validasi form menggunakan JavaScript
Proses data laporan menggunakan PHP
Tampilan sederhana dan mudah digunakan
